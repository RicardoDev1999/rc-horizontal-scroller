## RTL Support

The InfiniteSlider includes built-in support for Right-to-Left (RTL) languages. To enable RTL mode:

1. Set the `dir` attribute on your HTML or a parent element:

```html
<html dir="rtl">
  <!-- Your app content -->
</html>
```
